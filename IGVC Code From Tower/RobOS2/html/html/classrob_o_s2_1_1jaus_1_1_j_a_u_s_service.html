void robOS2.jaus.JAUSService.run ( )

Keeps the tread running. DO NOT call this method to start it up. Instead call the start() method. That will take care of all the checks ensuring that everything is up and running like it should be.

void robOS2.jaus.JAUSService.sendPacket ( DatagramPacket  packet)

Sends a UDP Packet out to the specified address and port in the IJAUSConfig file. This also checks to make sure the connection is open, and if it is not issues a fatal log4j message and simply returns from the function.

Parameters:
packet- Datagram packet to be sent out.
void robOS2.jaus.JAUSService.start ( )

Starts the JAUS Service. This will not only send new JAUS packets, it will also monitor for new packets coming from the COP

void robOS2.jaus.JAUSService.stop ( )

Stops the JAUS Service. No more packets will be sent after the service is stopped, and new incoming packets will be ignored.
